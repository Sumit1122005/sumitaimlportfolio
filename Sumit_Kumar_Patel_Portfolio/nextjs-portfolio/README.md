## ✨ Key Features
- **Modern Tech Stack**: Built with Next.js App Router for optimal performance and SEO.
- **Advanced Animations**: Fluid, physics-based UI transitions and staggered elements using Framer Motion.
- **Glassmorphism Aesthetic**: Custom noise textures, dynamic glowing gradients, and deep dark-mode UI for a premium "Apple-like" feel.
- **Interactive Bento-Box Grids**: Projects and Skills are displayed using a responsive, modern grid system with intuitive hover micro-interactions.
- **Zero-Backend Contact Form**: Securely utilizes mailto handlers for a frictionless messaging experience directly to `sumitkumarpatel01122005@gmail.com` without needing a complex backend.

## 🛠️ Tech Stack
- **Framework**: [Next.js (App Router)](https://nextjs.org/)
- **UI Library**: [React.js](https://reactjs.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: Custom Vanilla CSS with CSS Variables (`globals.css`)
- **Icons**: [Lucide React](https://lucide.dev/)

## ⚙️ Local Development & Setup

Follow these steps to run the portfolio locally on your machine.

### Prerequisites
Make sure you have Node.js (v18.0.0 or higher) installed.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name/nextjs-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open the app**
   Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
